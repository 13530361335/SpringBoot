# SpringBoot
SpringBoot-Demo项目
整理常用轮子


本地引包方式
```
        <dependency>
            <groupId>org.apache.ant</groupId>
            <artifactId>ant</artifactId>
            <version>1.8.2</version>
            <scope>system</scope>
            <systemPath>${project.basedir}/lib/ant-1.8.2.jar</systemPath>
        </dependency>
```
